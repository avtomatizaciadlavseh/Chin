HanziQuest (offline prototype)
==========================

Как открыть:
1) Распакуй архив.
2) Открой файл `index.html` двойным кликом (или через браузер).

Что внутри:
- Регистрация (localStorage) + 4 аватара (SVG, офлайн).
- Панель: XP, серия, прогресс.
- Урок по тонам: мини‑квиз (опционально синтез речи, если есть zh‑голос в браузере).
- Урок по иероглифам: мини‑квиз по смыслу/знаку.
- Квест про современный Китай: интерактивная история с лексикой и XP.
- Canvas‑тренажёр: рисование + подсказки + экспорт PNG.

Примечание:
Это учебный прототип. Для продакшна обычно добавляют SRS, базу HSK, аккаунты, аналитика, бэкенд, и т.д.
